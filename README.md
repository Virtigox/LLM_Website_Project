# Final Web Project AI

Three static pages introduce large language models, retrieval-augmented generation, and local deployment. Each page shares a neon-accented dark theme, floating navigation button, and accessible layout.

## Live Site
- https://virtigox.github.io/LLM_Website_Project/

## Pages
- **index.html** – LLM overview with marquee of ecosystem logos and theatre-style explainer video.
- **RAG-VectorDB.html** – Retrieval-augmented generation, vector stores, and pipeline walkthrough.
- **Localizing-Deployment.html** – Self-hosting benefits, tooling table, hardware tips, and centered video theatre.

## Stack
- HTML5 + semantic landmarks
- CSS custom properties with responsive clamps
- Vanilla JS for dropdown navigation and looping marquee

## Quick Start
1. Clone or download the repo.
2. Open any HTML file in a modern browser.
3. Ensure `logos/` and `pictures/` assets remain alongside the pages.
